Part One: Solidify Terminology

What is HTTP?

Http is the standard protocol used by servers and browsers to communicate with each other.

What is a URL

A URL is reference point to find content on the internet. It points to a particular location on the web

What is DNS?

DNS turns a URL into an IP address

What is a query string?

A query string provides extra information such as key words to search or page to navigate to.

What are two HTTP verbs and how are they different

Get and Post are two HTTP verbs. Get requests are used to retrieve information from a sight while POST is used to alter information on the site. Difference between looking at pictures on a site and posting your own picture onto the site.

What is an HTTP request?

HTTP request is when your browser points to a server and asks for some sort of information or action that is specified in the request.

What is an HTTP response?

This is the the reply the server sends to your browser. Usually the html. It will also send a status code (200, 404, etc.)

What is an HTTP header? Give a couple examples of request and response headers you have seen.

Part of a request or response that holds metadata and additional context. Preferred language is an example of this. Time that browser or server think it is is also one. This also can specify the preferred media formats.

What are the processes that happen when you type "http://somesite.com/some/page.html" into a browser?

  1. Browser makes a request at the somesite.com using http protocol
  2. somesite.com is turned into IP address indicating server using DNS
  3. The connection is made using default port (unless specified) which is 80
  4. the request is made to retreive /some/page.html
  5. The server responds with the either the page or with an error
  6. The information is loaded in the browser for the user
